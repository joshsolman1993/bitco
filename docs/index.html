🔌 API Documentation

Authentication Endpoints

POST /api/auth/register

Register a new user account

POST /api/auth/login

Login and receive JWT token

GET /api/auth/me

Get current user profile

Mining Endpoints

GET /api/miners

Get all miners for current user

POST /api/miners

Purchase a new miner

PUT /api/miners/:id/upgrade

Upgrade an existing miner

Market Endpoints

GET /api/market/prices

Get current market prices

POST /api/market/buy

Execute a buy order

POST /api/market/sell

Execute a sell order

WebSocket Events

// Connect to WebSocket
const ws = new WebSocket('ws://localhost:3001');

// Listen for tick updates
ws.on('tick', (data) => {
  console.log('Game tick:', data.tickNumber);
  console.log('Market prices:', data.marketPrices);
  console.log('Mining rewards:', data.miningRewards);
});

// Listen for market updates
ws.on('marketUpdate', (data) => {
  console.log('Price change:', data);
});

🚀 Quick Start Guide

For Developers

Installation
# Clone the repository
git clone https://github.com/yourusername/bitcoin-tycoon.git
cd bitcoin-tycoon

# Install backend dependencies
cd backend
npm install

# Setup database
npx prisma migrate dev
npx prisma db seed

# Start backend server
npm run dev

# In a new terminal, install frontend dependencies
cd ../frontend
npm install

# Start frontend development server
npm run dev
Environment Variables
# backend/.env
DATABASE_URL="postgresql://user:password@localhost:5432/bitcoin_tycoon"
JWT_SECRET="your-secret-key"
PORT=3001

# frontend/.env
VITE_API_URL=http://localhost:3001
VITE_WS_URL=ws://localhost:3001

Testing the API

PowerShell
# Run backend tests
cd backend
npm test

# Test API endpoints
.\test-api.ps1

# Test integration
.\test-integration.ps1
